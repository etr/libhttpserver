2. [x] **code-quality-reviewer** | `src/webserver.cpp:491` | error-handling
   stop() calls shutdown(impl_->bind_socket, 2) unconditionally. bind_socket is initialized to 0 in webserver_impl (webserver_impl.hpp:111) and is only set to a real value when the caller explicitly passes a pre-bound socket via create_webserver (constructor line 210). Also: daemon reset to nullptr after MHD_stop_daemon (finding 20). bind_socket now passed through webserver_impl constructor (finding 13).
